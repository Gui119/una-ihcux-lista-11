DESCRIÇÃO DAS TELAS

Tela 1 – Home
É a tela inicial do app. Mostra uma apresentação do Reciclo e explica rapidamente a ideia. Também tem um botão pra começar a usar.

Tela 2 – Filtros / Lista de itens
Aqui o usuário pode pesquisar itens (ex: eletrônicos, móveis, roupas). Também dá pra filtrar e ver avaliações dos itens disponíveis.

Tela 3 – Detalhes do item
Mostra todas as informações do item selecionado, como descrição, localização (ex: Contagem - MG), categoria e condições. Tem opção de demonstrar interesse.

Tela 4 – Cadastro do item (Doar)
Essa tela é pra quem quer doar algo. A pessoa preenche informações como nome do item, descrição, categoria e estado de conservação.

Tela 5 – Chat e Agendamento
Aqui o usuário conversa com outra pessoa para combinar a entrega/retirada do item. Também pode organizar o horário.

Tela 6 – Perfil do usuário
Mostra os dados do usuário, histórico e avaliações. Também tem uma parte de conquistas, tipo um incentivo pra pessoa usar mais o app.
